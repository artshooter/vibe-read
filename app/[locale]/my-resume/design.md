# 简历页面设计文档

## 1. 内容理解

这是一份前端工程师的个人简历，展示了：
- 核心主题：5年工作经验，全栈开发能力（Flutter + Web），AI 工具应用专家
- 目标读者：招聘方、技术面试官、潜在雇主
- 核心特色：
  - 垂直技术深度（从应用层到开源库二次开发）
  - AI 工具熟练应用（提效 20%，自动化工作流）
  - 注重实效（多个提效案例）
  - 开源贡献者

情绪基调：专业、自信、创新、注重效率和技术深度

## 2. 设计直觉

第一感受：
- **画面**：一条向上延伸的时间轴，从 2020 年到现在，象征职业成长轨迹
- **颜色**：深色背景 + 科技蓝/紫渐变（体现 AI 和现代科技感）
- **动态**：平滑滚动，卡片渐入效果，时间节点的脉动感
- **关键词**：专业、现代、科技感、AI 赋能、成长

脑海中的画面：
- 左侧是一条发光的时间轴线，从下到上延伸
- 右侧是工作经历和项目的卡片，随着滚动依次展现
- AI 相关的内容有特殊高亮效果（渐变或发光边框）
- 技能标签以流动的方式展示

## 3. 设计方案

### 3.1 视觉风格

**色彩方案**：
- 主背景：深色（#0a0a0a 或 #1a1a2e）
- 强调色：科技蓝到紫的渐变（#3b82f6 → #8b5cf6）用于 AI 相关内容
- 时间轴：淡蓝色发光线条
- 文字：白色主标题，灰色副文本
- 卡片：半透明深色背景，带微光边框

**字体**：
- 标题：较粗的无衬线字体（600-700 weight）
- 正文：清晰易读的无衬线字体（400-500 weight）
- 代码/技术关键词：等宽字体

**视觉元素**：
- 时间轴左侧垂直线条，节点用圆点标记
- 卡片使用圆角、阴影、半透明背景
- AI 相关内容用渐变边框或背景高亮
- 数据（如"提效 20%"）用大号数字 + 动画强调

**风格关键词**：现代、专业、科技感、AI 赋能、简洁

### 3.2 交互设计

**交互点**：

1. **时间轴交互**：
   - 滚动时时间轴节点依次点亮
   - 当前视口内的节点有脉动动画
   - 点击节点可快速跳转到对应内容

2. **项目卡片 Hover**：
   - Hover 时卡片微微上浮
   - 边框发光效果增强
   - 展示更多细节（如技术栈、成果数据）

3. **技能标签**：
   - 标签以流动方式展示
   - Hover 时放大并显示熟练度
   - 可点击筛选相关项目

4. **成就数据可视化**：
   - "提效 20%" 等数据用数字滚动动画
   - 进入视口时触发动画
   - 配合进度条或图表展示

5. **AI 工具展示**：
   - AI 相关内容有特殊渐变高亮
   - Hover 显示具体应用场景
   - 可展开查看详细工作流程

**实现方式**：
- 使用 Framer Motion 实现滚动触发动画
- Intersection Observer 监听元素进入视口
- CSS 渐变和模糊实现发光效果
- React hooks 管理交互状态

### 3.3 沉浸式设计

**核心隐喻**：职业成长时间线

**导航逻辑**：
- 纵向滚动为主
- 左侧时间轴始终可见，作为导航锚点
- 内容按时间倒序排列（最新在上）

**滚动体验**：
- 平滑滚动，带缓动效果
- 卡片随滚动依次渐入
- 视差效果：背景和内容以不同速度滚动
- 时间节点与内容同步高亮

**场景营造**：
- 深色背景营造专业、沉浸的阅读环境
- 发光元素（时间轴、AI 高亮）营造科技感
- 动画节奏不急不慢，让读者专注于内容

### 3.4 响应式设计

**桌面端（≥1024px）**：
- 左侧固定时间轴（20% 宽度）
- 右侧内容区域（80% 宽度）
- 卡片最大宽度限制，居中展示

**平板端（768px - 1023px）**：
- 时间轴缩小到左侧窄条
- 内容区域占主要空间
- 卡片宽度自适应

**移动端（<768px）**：
- 时间轴变为顶部固定横向进度条
- 内容全宽展示
- 卡片垂直堆叠
- 简化动画效果，提升性能

**触摸优化**：
- 增大可点击区域
- 移除纯 hover 效果，改为点击展开
- 手势滑动支持

## 4. 技术实现

### 核心组件

1. **Hero 组件**：
   - 个人信息展示（姓名、联系方式）
   - 自我评价摘要
   - 支持 `inHome` 模式（首页卡片）

2. **Timeline 组件**：
   - 垂直时间轴渲染
   - 节点标记和连接线
   - 滚动同步高亮

3. **ExperienceCard 组件**：
   - 工作经历卡片
   - Hover 交互效果
   - 渐入动画

4. **SkillCloud 组件**：
   - 技能标签云
   - 流动布局
   - 交互筛选

5. **AchievementMetric 组件**：
   - 成就数据展示
   - 数字滚动动画
   - 进度条可视化

6. **ProjectCard 组件**：
   - 个人项目卡片
   - 翻转或展开效果
   - AI 项目特殊高亮

### 依赖库

- **framer-motion**：动画效果
- **react-intersection-observer**：滚动触发
- **clsx / tailwind-merge**：样式管理

### 性能考虑

- 动画使用 `transform` 和 `opacity`，避免触发重排
- 移动端简化动画，减少资源消耗
- 图片懒加载（如果有头像或项目截图）
- 使用 CSS `will-change` 优化动画性能

## 5. 首页封面设计

**Cover 组件**（h-[400px]）：
- 深色渐变背景（蓝紫渐变）
- 居中显示姓名和职位
- 底部展示核心标签（AI 工具、全栈、开源贡献）
- 整体包裹在 `<Link>` 中，可点击跳转
- Hover 时有微妙的放大和发光效果

## 6. 内容组织

**页面结构**：
1. Hero 区域 - 个人信息和自我评价
2. 技能概览 - 技能标签云
3. 工作经历 - 时间轴 + 经历卡片
   - Matrixport（2022.05 - 2025.06）
   - SHEIN（2020.09 - 2022.05）
4. 个人项目 - 项目卡片（AI 相关高亮）
5. 教育经历 - 简洁展示

**AI 内容高亮策略**：
- AI 工具应用：蓝紫渐变边框
- 提效数据：大号数字 + 动画
- 开源贡献：特殊徽章标记
